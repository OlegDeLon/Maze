Руководство Maze

  • В данном проекте можно познакомиться с лабиринтами и пещерами, а также основными алгоритмами их обработки, такими как: генерация, отрисовка, поиск решения.

    • Описание программы:
      1. В программе реализован графический пользовательский интерфейс на базе GUI-библиотеки Qt
      2. Генерируется лабиринт согласно алгоритму Эллера
      3. Генерируется невероятная пещера по заданным параметрам
      4. Толщина "стены" и линия решения - 2 пикселя
      5. Загруженный лабиринт отрисован на экране в поле размером 500 x 500 пикселей
      6. Максимальный размер лабиринта - 50х50
      7. В программе предусмотрена кнопка для загрузки лабиринта из файла

    • Поддерживаемые функции:
      1. Генерация идеального лабиринта
      2. Решение лабиринта
      3. Генерация пещер

  • Чтобы установить программу выполните make install

  • Чтобы удалить программу выполните make uninstall
